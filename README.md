Online File sharing application.

REST APIs & Node.js used for backend.
React + TS frontend
Peer.Js for P2P Connection


This application has password encryptions (bcrypt), access + refresh tokens (jsonwebtoken) for registered accounts, and file encryption. Better-sqlite3 used for database queries.

Frontend consists of account registration, login, and the page to connect to another peer via ID and send a file.

Hosted on AWS EC2.


https://main.p2p-fs.online/
